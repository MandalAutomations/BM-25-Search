# REST endpoints for the GitHub Marketplace API

To help manage your app on GitHub Marketplace, use these GitHub Marketplace API endpoints.

> \[!NOTE]
> This article applies to publishing apps in GitHub Marketplace only. For more information about publishing GitHub Actions in GitHub Marketplace, see [Publishing actions in GitHub Marketplace](/en/actions/creating-actions/publishing-actions-in-github-marketplace).

Here are some useful endpoints available for Marketplace listings:

* [List plans](/en/rest/apps/marketplace#list-plans)
* [List accounts for a plan](/en/rest/apps/marketplace#list-accounts-for-a-plan)
* [Get a subscription plan for an account](/en/rest/apps/marketplace#get-a-subscription-plan-for-an-account)
* [List subscriptions for the authenticated user](/en/rest/apps/marketplace#list-subscriptions-for-the-authenticated-user)

See these pages for details on how to authenticate when using the GitHub Marketplace API:

* [Authorization options for OAuth apps](/en/apps/oauth-apps/building-oauth-apps/authorizing-oauth-apps)
* [Authentication options for GitHub Apps](/en/apps/creating-github-apps/authenticating-with-a-github-app/about-authentication-with-a-github-app)

> \[!NOTE]
> [Rate limits for the REST API](/en/rest/using-the-rest-api/rate-limits-for-the-rest-api) apply to all GitHub Marketplace API endpoints.
