# Activity dashboard

The Activity dashboard gives you an overview of all the activity in your enterprise.

The Activity dashboard provides weekly, monthly, and yearly graphs of the number of:

* New pull requests
* Merged pull requests
* New issues
* Closed issues
* New issue comments
* New repositories
* New user accounts
* New organizations
* New teams
