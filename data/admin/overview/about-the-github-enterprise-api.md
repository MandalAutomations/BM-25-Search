# About the GitHub Enterprise API

GitHub Enterprise Server supports REST and GraphQL APIs.

With the APIs, you can automate many administrative tasks. Some examples include:

* Perform a configuration run to apply changes to Management Console settings. For more information, see [REST API endpoints for managing GitHub Enterprise Server](/en/enterprise-server@3.20/rest/enterprise-admin/manage-ghes).

* Configure LDAP sync. For more information, see [REST API endpoints for LDAP](/en/enterprise-server@3.20/rest/enterprise-admin/ldap).

* Collect statistics about your enterprise. For more information, see [REST API endpoints for admin stats](/en/enterprise-server@3.20/rest/enterprise-admin/admin-stats).

* Manage your enterprise account. For more information, see [Managing enterprise accounts](/en/enterprise-server@3.20/graphql/guides/managing-enterprise-accounts).

For the complete documentation for the GitHub Enterprise Server APIs, see [GitHub REST API](/en/enterprise-server@3.20/rest) and [GitHub GraphQL API](/en/enterprise-server@3.20/graphql).
