# Disabling GitHub Connect

You can disable GitHub Connect to stop the connection from GitHub.com to GitHub Enterprise Cloud.

When you disconnect from GitHub Enterprise Cloud, the GitHub Connect GitHub App is deleted from your enterprise account and credentials stored on your GitHub Enterprise Server instance are deleted.

1. In the top-right corner of GitHub Enterprise Server, click your profile picture, then click **Enterprise settings**.
1.
