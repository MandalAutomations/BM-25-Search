# Setting up GitHub Copilot for yourself

Start using GitHub Copilot to write code faster on GitHub.

## Get access to GitHub Copilot

Copilot Free gives you up to 2,000 code completions and 50 chat messages per month at no cost. [Start using Copilot Free](https://github.com/copilot?ref_product=copilot\&ref_type=engagement\&ref_style=text\&ref_plan=free).

For higher limits and access to all available models, subscribe to **Copilot Pro** or **Copilot Pro+**. See [Getting started with a GitHub Copilot plan](/en/copilot/how-tos/manage-your-account/get-started-with-a-copilot-plan) for details on all plans, including free options for students, teachers, and open source maintainers.

## Configure settings (optional)

If you have your own Copilot plan, manage your policies and feature access from your [Copilot settings](https://github.com/settings/copilot?ref_product=copilot\&ref_type=engagement\&ref_style=text). You can also enable Copilot Memory so Copilot remembers useful context about your repositories.

## Set up personal custom instructions (optional)

Personal custom instructions tell Copilot Chat about your preferences — such as your preferred language, coding style, or response format. To add them, open any Copilot Chat conversation on GitHub, click your profile picture, and select **Personal instructions**. For more information, see [Adding personal custom instructions for GitHub Copilot](/en/copilot/how-tos/configure-custom-instructions/add-personal-instructions).
