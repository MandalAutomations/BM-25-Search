# Managing volume licenses for GitHub Advanced Security

You can monitor and control the availability and consumption of licenses for Advanced Security in repositories in your enterprise.

There are two different ways to pay for GitHub Secret Protection, GitHub Code Security, and GitHub Advanced Security licenses: volume/subscription licenses purchased in advance or usage-based metered billing paid in arrears. This article is about volume/subscription licenses. For information about the two different billing models, see [GitHub Advanced Security license billing](/en/billing/managing-billing-for-your-products/managing-billing-for-github-advanced-security/about-billing-for-github-advanced-security).

For information about using policies to control use of licenses in your enterprise, see [Enforcing policies for code security and analysis for your enterprise](/en/enterprise-cloud@latest/admin/policies/enforcing-policies-for-your-enterprise/enforcing-policies-for-advanced-security-in-your-enterprise).

## Changing the size of your license

1. Navigate to your organization or enterprise. For example, from the [Organizations](https://github.com/settings/organizations?ref_product=github\&ref_type=engagement\&ref_style=text) or [Enterprises](https://github.com/settings/enterprises?ref_product=ghec\&ref_type=engagement\&ref_style=text) pages on GitHub.com.

2. From the list of "Billing & licensing" pages, click **Licensing**.

   ![Screenshot of the Advanced Security licensing screen. The "Manage licenses" button is outlined in orange.](/assets/images/help/enterprises/ghas-licenses-dropdown.png)

3. To add new licenses, select <svg version="1.1" width="16" height="16" viewBox="0 0 16 16" class="octicon octicon-kebab-horizontal" aria-label="Open menu" role="img"><path d="M8 9a1.5 1.5 0 1 0 0-3 1.5 1.5 0 0 0 0 3ZM1.5 9a1.5 1.5 0 1 0 0-3 1.5 1.5 0 0 0 0 3Zm13 0a1.5 1.5 0 1 0 0-3 1.5 1.5 0 0 0 0 3Z"></path></svg>, then click **Manage licenses**.

4. Under "Total licenses", click the plus or minus buttons to add or remove licenses.

   ![Screenshot of the Advanced Security license screen. A text box with the number 5, with a minus and a plus button, are outlined in orange.](/assets/images/help/enterprises/ghas-add-licenses.png)

5. Click **Confirm licenses**.

## Canceling your Advanced Security subscription

1. Navigate to your organization or enterprise. For example, from the [Organizations](https://github.com/settings/organizations?ref_product=github\&ref_type=engagement\&ref_style=text) or [Enterprises](https://github.com/settings/enterprises?ref_product=ghec\&ref_type=engagement\&ref_style=text) pages on GitHub.com.
2. From the list of "Billing & licensing" pages, click **Licensing**.
3. To the right of "Advanced Security", select <svg version="1.1" width="16" height="16" viewBox="0 0 16 16" class="octicon octicon-kebab-horizontal" aria-label="Open menu" role="img"><path d="M8 9a1.5 1.5 0 1 0 0-3 1.5 1.5 0 0 0 0 3ZM1.5 9a1.5 1.5 0 1 0 0-3 1.5 1.5 0 0 0 0 3Zm13 0a1.5 1.5 0 1 0 0-3 1.5 1.5 0 0 0 0 3Z"></path></svg>, then click **Cancel subscription**.
4. To confirm your cancellation, click **I understand, cancel Advanced Security**.

## Disabling GitHub Advanced Security in an enterprise

Enterprise owners can disable GitHub Advanced Security completely and set a policy to prevent future re-enablement. See [Disabling GitHub Advanced Security for your enterprise](/en/billing/how-tos/products/disable-ghas-for-enterprise).
