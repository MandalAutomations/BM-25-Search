# Profile

Get started with your GitHub profile.

## About your GitHub profile

Your profile is how other people discover and learn about you on GitHub. You can customize your profile to highlight your best work and share more about yourself.

* For a general tutorial on personalizing your profile, see [Personalize your profile](/en/account-and-profile/tutorials/personalize-your-profile).
* For more specific profile customization, see [How-tos for your GitHub account and profile](/en/account-and-profile/how-tos/setting-up-and-managing-your-github-profile).
* For reference information, see [Profile reference](/en/account-and-profile/reference/profile-reference).
