# Your organization's profile

Your organization's profile page shows basic information about your organization.

You can customize your organization's profile by adding any of the following:

* A description
* A location
* A website
* An email address

You can also pin important repositories and add a `README.md` file to help orient visitors. For more information, see [Customizing your organization's profile](/en/organizations/collaborating-with-groups-in-organizations/customizing-your-organizations-profile).

Organizations that use GitHub Enterprise Cloud can confirm their organization's identity and display a "Verified" badge on their organization's profile page by verifying the organization's domains with GitHub. For more information, see [Verifying or approving a domain for your organization](/en/organizations/managing-organization-settings/verifying-or-approving-a-domain-for-your-organization) in the GitHub Enterprise Cloud documentation.
