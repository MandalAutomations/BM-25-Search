# About enterprise security

Discover the security features available to your enterprise.

GitHub has many features that help you improve and maintain the quality of your code. Some of these are included in all plans, such as dependency graph and Dependabot alerts. Other security features require an Advanced Security license to run on repositories apart from public repositories on GitHub.com.

To learn about the security features available to your enterprise, see [Security and code quality documentation](/en/enterprise-cloud@latest/code-security).

To learn about the extra security features available with an Advanced Security license, see [About GitHub Advanced Security](/en/enterprise-cloud@latest/get-started/learning-about-github/about-github-advanced-security).
