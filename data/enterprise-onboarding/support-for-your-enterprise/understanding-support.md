# Understanding enterprise support

Explore the various GitHub Support offerings available to you.

## About GitHub Support

Standard GitHub Support is provided as part of your GitHub Enterprise subscription and offers a comprehensive set of support and management features tailored for large businesses and teams. See [About GitHub Support](/en/enterprise-cloud@latest/support/learning-about-github-support/about-github-support).

GitHub Enterprise customers can also purchase GitHub Premium Support.

### About GitHub Premium Support

GitHub Premium Support is a **paid**, supplemental support offering that is ideal for organizations requiring higher priority support, faster response times, dedicated account management, and 24/7 comprehensive support. In addition to all of the benefits of standard GitHub Enterprise Support, GitHub Premium Support offers:

* 24/7 availability: Ensures that support is available at any time, day or night, including holidays, which is crucial for organizations operating globally or with mission-critical applications.

* Faster response and resolution times: Accelerated response times for support requests mean that critical issues are addressed and resolved more quickly, minimizing downtime and disruption.

* Escalation management: Dedicated escalation management ensures that complex issues are addressed promptly and efficiently, with a clear path for escalating critical incidents.

* Early access to new features: Participation in preview programs and early access to new features allows you to stay ahead of the curve with the latest GitHub innovations.

There are two support plans available with GitHub Premium Support:

* **Premium plan**: Includes priority ticket handling, access to premium content, and more.

* **Premium Plus plan/Mission Critical Services for GitHub plan**: Includes all Premium Plan benefits plus additional features such as named Customer Reliability Engineer (CRE), quarterly enhanced health checks, technical advisory hours, and application upgrade assistance.

For detailed information about GitHub Premium Support and the two support plans, see [About GitHub Premium Support](/en/enterprise-cloud@latest/support/learning-about-github-support/about-github-premium-support).

## Contacting GitHub Support

To contact GitHub Support, visit the [GitHub Support portal](https://support.github.com/). For more information, see [Creating a support ticket](/en/enterprise-cloud@latest/support/contacting-github-support/creating-a-support-ticket).

Anyone can use the [GitHub Support portal](https://support.github.com/) to view and manage support tickets about GitHub, but there are special steps to follow before using the [GitHub Support portal](https://support.github.com/) to create tickets about an enterprise account. See [Getting your enterprise started with the GitHub Support portal](/en/enterprise-cloud@latest/support/contacting-github-support/getting-your-enterprise-started-with-the-github-support-portal).
