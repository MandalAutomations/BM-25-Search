# Enabling repository search across environments

Unify search results in GitHub Enterprise Server by connecting your account on GitHub.com or GHE.com.

You can search for designated private repositories on GitHub Enterprise Cloud from your GitHub Enterprise Server instance. For more information about searching across environments, see [About searching on GitHub](/en/enterprise-server@3.20/search-github/getting-started-with-searching-on-github/about-searching-on-github#searching-repositories-on-githubcom-from-your-private-enterprise-environment).

## Prerequisites

An enterprise owner for GitHub Enterprise Server must enable GitHub Connect and unified search for private repositories. For more information, see [Enabling unified search for your enterprise](/en/enterprise-server@3.20/admin/configuration/configuring-github-connect/enabling-unified-search-for-your-enterprise).

## Enabling unified repository search from GitHub Enterprise Server

1. Sign in to both your user account on GitHub Enterprise Server **and** your user account on GitHub Enterprise Cloud (GitHub.com or GHE.com).
2. On GitHub Enterprise Server, in the upper-right corner of any page, click your profile picture, then click **Settings**.

![Screenshot of a user's account menu on GitHub. The menu item "Settings" is outlined in dark orange.](/assets/images/help/settings/userbar-account-settings-global-nav-update.png)

1. In the left sidebar, click **GitHub Connect**.
2. Click **Connect to GitHub.com**.
